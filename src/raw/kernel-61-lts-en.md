## Kernel 6.1 LTS: What's New

The Linux 6.1 kernel marks a turning point for the LTS series. The main new features include improved hardware support and performance optimizations for server workloads.

### Key Points

- Better memory management for high-density systems
- Improved Btrfs support with more efficient checksums
- New drivers for datacenter NICs
