# thelinuxformula

Repository contenente il codice sorgente del sito [thelinuxformula.com](https://thelinuxformula.com).

## Demo

🔗 **Live Demo:** https://lucasacchiricciardi.github.io/the-linux-formula

## Tech Stack

- HTML5 + Tailwind CSS (CDN)
- Vanilla JavaScript (zero framework)
- Web Worker per async fetching
- LZ-string compression per localStorage
- No build step richiesto (vanilla JS)
